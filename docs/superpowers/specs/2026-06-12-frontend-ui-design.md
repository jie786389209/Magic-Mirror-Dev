# Magic Mirror Dev — 前端界面设计规格

> 日期：2026-06-12 | 状态：已确认 | 模块：全局 UI

---

## 1. 设计目标

构建一个**平台级 AI 开发助手**前端，核心体验为聊天为主，融合知识库、工具、Skill、记忆等模块管理能力。视觉风格参考 VS Code/Cursor 的深色 IDE 沉浸体验。

---

## 2. 技术方案

| 技术 | 版本 | 用途 |
|------|------|------|
| Vue 3 | 3.4 | 前端框架 |
| Element Plus | 2.7 | UI 组件库 |
| Pinia | - | 状态管理 |

## 3. 整体布局

```

│                     顶栏（48px）                          │
│  [≡]  Magic Mirror Dev    [当前会话标题]    [⚡] [🔔] [👤] │
├────┬────────────────────────────────────┬─────────────────┤
│ 🏠 │         中央对话区                  │   右侧面板       │
│ 💬 │         （主工作区）                 │   （可切换）     │
│ 📁 │                                    │                 │
```

## 6. 消息交互

### 消息气泡

- **用户消息**：右对齐，背景 `#1E293B`，右侧绿色 3px 边框 `#22C55E`
- **AI 消息**：左对齐，透明背景，左侧灰色 3px 边框 `#334155`，绿色头像圆
- **流式输出**：AI 消息逐字渲染，末尾绿色闪烁光标 `▊`，完成后消失
- **消息操作栏**：hover 显示 — 复制 / 重新生成 / 点赞/踩
- **用户消息可编辑**：点击铅笔图标，修改后重新发送

### 输入区

| 功能 | 触发 | 说明 |
|------|------|------|
| 命令面板 | `/` 或 Ctrl+/ | 弹出命令列表 |
| 文件引用 | `@` | 引用知识库文档 |
| 附件上传 | 📎 或拖拽 | 图片/文档 |
| 发送 | Enter | Ctrl+Enter 换行 |
| 停止生成 | 发送按钮变停止 | 中断流式输出 |

### 状态处理

| 状态 | 处理 |
|------|------|
| 加载中 | 骨架屏 |
| 空状态 | Logo + "输入问题开始对话" |
| 错误 | Toast 通知 + 消息内联错误 |
| 网络断开 | 黄色提示条 "连接已断开，正在重连..." |
| 流式中断 | 灰色标记 "[生成已中断]"，可继续 |

## 7. 右侧上下文面板

- 宽度：240-480px（可拖拽）
- Tab 切换：文件 / 记忆 / 工具 / 上下文
- 每个 Tab 为后续模块预留扩展点
- 点击 ⚡ 一键折叠/展开

## 8. 响应式策略

| 断点 | 布局 |
|------|------|
| ≥1440px | 三栏完整 |
| 1024-1440px | 右侧面板默认收起 |
| 768-1024px | 左侧图标栏无文字 |
| <768px | 图标栏移至底部导航 |

## 9. 项目结构（前端）

```
src/
├── App.vue
├── main.ts
├── assets/
│   └── styles/
│       ├── variables.css      # CSS 变量（色板、字体、间距）
│       ├── reset.css           # 全局重置
│       └── transitions.css     # 过渡动画
├── components/
│   ├── layout/
│   │   ├── AppLayout.vue       # 顶层三栏布局
│   │   ├── TopBar.vue          # 顶栏
│   │   ├── SideIconBar.vue     # 左侧图标栏
│   │   └── RightPanel.vue      # 右侧上下文面板
│   ├── chat/
│   │   ├── ChatView.vue        # 对话主视图
│   │   ├── MessageList.vue     # 消息列表
│   │   ├── MessageBubble.vue   # 单条消息气泡
│   │   ├── StreamCursor.vue    # 流式光标动画
│   │   └── ChatInput.vue       # 输入区（含命令面板）
│   └── common/
│       ├── SkeletonLoader.vue  # 骨架屏
│       └── EmptyState.vue      # 空状态占位
├── views/
│   ├── DashboardView.vue       # 首页
│   ├── KnowledgeView.vue       # 知识库（后续）
│   ├── ToolsView.vue           # 工具（后续）
│   ├── SkillsView.vue          # Skill（后续）
│   ├── MemoryView.vue          # 记忆（后续）
│   └── SettingsView.vue        # 设置
├── stores/
│   ├── chat.ts                 # 对话状态
│   ├── app.ts                  # 全局 UI 状态
│   └── settings.ts             # 设置状态
├── router/
│   └── index.ts
├── api/
│   ├── client.ts               # HTTP 客户端（axios + SSE）
│   └── chat.ts                 # 对话 API
└── utils/
    ├── markdown.ts             # Markdown 渲染
    └── constants.ts            # 常量
```
